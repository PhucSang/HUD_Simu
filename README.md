# HUD Simu

Testing basic functionality.

## Installation

1. Mở SillyTavern
2. Đi đến Extensions → Install Extension (Cài đặt Tiện ích mở rộng)
3. Dán đường dẫn GitHub sau vào: `https://github.com/PhucSang/HUD_Simu`
4. Nhấn Install và đợi quá trình hoàn tất.
5. Refresh (Tải lại) trang web.

## Testing

Tìm "HUD Simu" trong tab cài đặt Extensions (cột bên phải).
